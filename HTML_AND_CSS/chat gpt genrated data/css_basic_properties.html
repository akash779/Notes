CSS Basic Properties

Background

The background properties allow you to customize the background of an element.

  • background-color: sets the background color.
  • background-image: sets the background image.
  • background-attachment: sets whether the background scrolls with the content or remains fixed.
  • background-repeat: sets how the background image is repeated.
  • background-size: sets the size of the background image.

Example:

            body {
                background-color: #f5f5f5;
                background-image: url('background.jpg');
                background-attachment: fixed;
                background-repeat: no-repeat;
                background-size: cover;
            }

Fonts

The font properties allow you to control the appearance of text.

  • font-family: sets the font family.
  • font-style: sets the font style (e.g., italic).
  • font-size: sets the font size.
  • font-weight: sets the font weight (e.g., bold).

Example:

            p {
                font-family: 'Arial', sans-serif;
                font-style: italic;
                font-size: 18px;
                font-weight: bold;
            }

  • color: sets the text color.

  • text-transform: transforms the case of the text (e.g., uppercase, lowercase).

Example:

                word-spacing: 5px;
            }

Shorthand

CSS provides shorthand notations to simplify the declaration of multiple properties.

  • background: combines background properties into a single declaration.
  • font: combines font properties into a single declaration.
  • text: combines text properties into a single declaration.

Example:

            body {
                background: #f5f5f5 url('background.jpg') fixed no-repeat cover;
                font: italic bold 18px/1.5 'Arial', sans-serif;
                text: center justify underline uppercase 20px/1.5 #333;
            }
