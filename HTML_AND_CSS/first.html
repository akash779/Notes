Responsive Notes

This website demonstrates how to make a responsive website using CSS media queries.

Syntax:

    @media only screen and (max-width: 600px) {
      /* Styles for screens with a maximum width of 600px */
      /* CSS code for small screens */
    }
    @media only screen and (min-width: 601px) and (max-width: 900px) {
      /* Styles for screens with a minimum width of 601px and maximum width of 900px */
      /* CSS code for medium screens */
    }
    @media only screen and (min-width: 901px) {
      /* Styles for screens with a minimum width of 901px */
      /* CSS code for large screens */
    }
