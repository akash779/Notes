Exploring the HTML target Attribute

Example 1: _blank

Visit Example.com

Opens the link in a new browser tab or window.

Example 2: _self (Default)

Visit Example.com

Opens the link in the same frame or tab.

Example 3: _parent

Visit Example.com

Opens the link in the parent frame or window.

Example 4: _top

Visit Example.com

Breaks out of all frames and displays the link in the top-level window or tab.

Example 5: Custom Frame Name

Visit Example.com in My Frame

Opens the link in a custom-named frame or window. If it doesn't exist, a new one is created.
